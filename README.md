# Cloud-Based Event Registration System

## 📖 Project Description

The Cloud-Based Event Registration System is a web application that enables users to register for events online. It uses HTML, JavaScript, and Firebase to validate inputs and store data in real time, making the system efficient, secure, and accessible from anywhere.

## 🚀 Features

* User registration form
* Input validation using JavaScript
* Real-time data storage using Firebase
* Simple and user-friendly interface

## 🛠️ Technologies Used

* HTML (with inline CSS)
* JavaScript
* Firebase

## ⚙️ How to Run the Project

1. Download or clone this repository
2. Open the project folder
3. Open `index.html` in your browser

## 🔥 Firebase Setup

1. Go to Firebase Console
2. Create a new project
3. Enable Realtime Database or Firestore
4. Copy Firebase configuration
5. Paste it into your `script.js` file

## 📊 Project Workflow

1. User enters details in form
2. Data is validated using JavaScript
3. Data is sent to Firebase
4. Data is stored in database
5. Confirmation is shown to user
